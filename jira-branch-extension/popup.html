📖 Hướng dẫn lấy API Token
1 Vào bitbucket.org → click avatar góc phải → Personal Bitbucket settings
⚠️ Không phải "Account settings"
2 Lấy email: Sidebar → Account settings → xem dòng "Your Atlassian account address is: ..."
3 Tạo token: Sidebar → API tokens → Create API token
4 Tick scope → Create → copy token (chỉ hiện 1 lần)
✅ Repositories → Read
✅ Repositories → Write
Không cần tick Account hay Workspace
→ Mở trang tạo API Token ↗
Email Atlassian — xem bước 2 ở trên
Token chỉ hiện 1 lần khi tạo — copy và paste vào đây
