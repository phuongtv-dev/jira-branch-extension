🪣 Bitbucket · API Token
📖 How to get a Bitbucket API Token
1 Go to bitbucket.org → click avatar → Personal Bitbucket settings
⚠️ Not "Account settings" — that goes to Atlassian
2 Find your email: Sidebar → Account settings → look for "Your Atlassian account address is:"
3 Sidebar → API tokens → Create API token → select scopes:
✅ Repositories → Read
✅ Repositories → Write
✅ Workspace → Read
read:repository:bitbucket
write:repository:bitbucket
read:workspace:bitbucket
4 Click Create → copy the token immediately (shown only once)
→ Open API Tokens page ↗
Token is shown only once — copy it right after creating

🐙 GitHub · Personal Access Token
📖 How to get a GitHub Personal Access Token
1 Go to github.com → click your avatar (top right) → Settings
2 Scroll down → Developer settings → Personal access tokens → Tokens (classic)
3 Click Generate new token (classic) → set a name and expiry → select scope:
✅ repo (Full control of private repositories)
4 Click Generate token → copy immediately (shown only once)
→ Open GitHub Token page ↗
Optional — will be auto-detected from token
Starts with ghp_ — copy it right after creating, shown only once
